# 一些分析

## 实验

- 1、直接baseline：14W
- 2、baseline把100w个数据改成200w个数据：13W
- 3、处理100种运单：4.3W
- 4、处理1000种运单：4.5W
  - 增加运单却mse增加？
    - 1、可能是迭代的次数不够，没训练到位
    - 2、有很多垃圾数据影响(可能性较大)
- 5、处理20种运单：3.6W
  - 迷惑？
    - 1、垃圾数据太多？
- 6、处理“下采样数据版本1”，-->16894条数据：5.2W
- 7、把6的mse改成越小越好，并迭代2000次：23W
- 8、把7改成迭代100次：22W
